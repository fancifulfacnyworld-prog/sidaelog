클레르 옵스퀴르 33원정대는 하나의 감정을 유지하기 위해 굉장히 세심하게 설계된 게임처럼 느껴졌다.
<br/>
이 게임은 끊임없이 죽음과 무력감을 이야기한다.<br/>
배경 그래픽, 음악, 몬스터 디자인, 전투 방식, UI까지.<br/>
분석한 모든 갈래가 결국 같은 감정을 향해 움직이고 있었다.
<br/>
가장 먼저 눈에 들어오는 건 거대한 배경 그래픽이다.<br/>
우울한 색감과 중압감이 느껴지는 풍경들.<br/>
이 세계 속 인간은 굉장히 작고 무력한 존재처럼 보인다.
<br/>
게임의 스토리 또한 인간의 소멸을 다루고 있기 때문에,<br/>
유저는 자연스럽게 세계 전체에 깔린 멸망의 분위기를 받아들이게 된다.
<br/>
몬스터 디자인 또한 비슷하다.<br/>
기괴하지만 단순히 혐오스럽게 느껴지진 않는다.<br/>
아름답지만 죽어가는 세계와 어울리는 방향으로 통일되어 있다.
<br/>
흥미로웠던 건 플레이 방식이었다.
<br/>
웅장하고 감정적인 세계관이라면 실시간 액션이 더 어울릴 것 같지만,<br/>
클레르 옵스퀴르는 턴제를 선택했다.
<br/>
덕분에 빠르게 전투를 소비하기보다, 잠시 멈춰 상황과 분위기를 바라보게 된다.<br/>
전투 중에도 캐릭터들의 위태로운 분위기와 죽음의 감각이 계속 유지된다.<br/>
평소 전투는 큰 압박감 없이 진행되지만,<br/>
중요한 전투에서는 긴장감을 급격하게 끌어올린다.
<br/>
보스전에서 대부분 보스가 먼저 공격을 시작하는 구조 또한<br/>
플레이어를 무력한 입장에 놓이게 만든다.
<br/>
UI 역시 같은 방향으로 설계되어 있다.
<br/>
연출은 굉장히 과감하지만 이상하게 과하다는 느낌은 들지 않는다.<br/>
배경 그래픽, 음악, 세계관 분위기와 자연스럽게 어우러져 있기 때문이다.
<br/>
이 게임의 UI는 단순히 정보를 전달하는 역할보다, 긴장감과 분위기를 유지시키는 장치에 가깝게 느껴졌다.
<br/>
클레르 옵스퀴르의 세계는 빠르게 지나가기 아깝다.<br/>
AI로 생성된 이미지처럼 기괴하게 멈춰 있는 풍경들,<br/>
우울하면서도 웅장한 거대한 세계는 플레이어를 계속 그 자리에 붙잡아 둔다.
<br/>
게임은 플레이어를 쉬지 않고 달리게 하기보다,그 세계를 오래 바라보게 만든다.
<br/>
그 과정에서 플레이어는<br/>
아주 거대한 세계 속 티끌 같은 존재가 된 것 같은 감각을 느끼게 된다.
<br/>
무기력함,<br/>
압도감,<br/>
그리고 다가오는 죽음.
<br/>
클레르 옵스퀴르 33원정대는 이런 감정들을 굉장히 세심하게 하나로 연결하고 있다.
