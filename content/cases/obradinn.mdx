UI로 게임을 읽는다고 하면,<br/>
믿겠어?
<QuestionBlock>
왜 사건은 죽음에서부터 시작할까?<br/>
UI가 화면에서 사라져야 했던 이유는 뭘까?<br/>
왜 기억을 조각조각 보여줄까??<br/>
</QuestionBlock>
오브라딘호의 귀환에는 UI가 거의 없다.
<br/>
등장인물의 표정과 시선,<br/>
짧은 대화와 주변 소리만 남는다.
<br/>
게임이 설명해 주는 게 없어<br/>
나는 계속 장면을 관찰하게 됐다.
<br/>
보통 추리물은<br/>
용의자와 피해자가 생기면서 감정이 치우치기 마련이다.<br/>
그런데 오브라딘호의 귀환은 달랐다.
<br/>
누군가를 미워하거나 의심하다가도 <br/>
금새 다른 기억을 관찰하게 됐다.
<br/>
플레이를 하며 이런 질문이 남았다.
