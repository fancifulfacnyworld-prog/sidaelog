사건은 죽음에서부터 시작한다.<br/>
유저가 사건의 현장에 처음부터 같이 있었다면 더 긴박하고<br/>
생동감있게 게임을 즐길 수 있지 않나? 
<br/>
오브라딘 호에 탄 모든 선원들이 죽어서 돌아온 걸로 추정되는데 왜 게임은<br/>
탐정 역할을 유저에게 부여하지 않았을까?
<br/>
사건이 일어나는 그 순간에 시작한다면 우리가 느낄 감정은 혼란스러움과 복잡함이다.<br/>
등장인물을 구조해야 하는가? 등장인물과 같이 행동해야 하는가?<br/>
이 게임은 그걸 원하지 않았다.<br/>
그래서 이미 끝난 사건, 죽음에서 시작한다.
<br/>
유저는 자연스럽게 관찰자의 위치에 놓이게 된다.
<br/>
우리가 수석 조사관이 된 이유도 비슷하다.<br/>
탐정은 선과 악을 나눈다. <br/>
유저의 정의감을 건드는 선택이다. 하지만 조사관은 어떨까?
<br/>
조사관은 제 3자의 입장에서 사실을 찾아 사건을 관찰할 뿐 다른 감정을 느낄 필요가 없어진다.<br/>
틀리는 것 조차 관찰의 과정으로 판단해 패널티도 적용하지 않았다.
<br/>
오브라딘호의 귀환은 <br/>
관찰에 방해되는 감정을 최대한 덜어내려 한 것 같다.<br/>
UI또한 화면에 녹아들어 없는 것처럼 작동한다.
<br/>
그 덕분에 장면 관찰이 굉장히 중요하다는 걸 자연스레 체감하게 된다.
